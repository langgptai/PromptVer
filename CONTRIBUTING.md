# 贡献指南

感谢关注 PromptVer！

## 如何贡献

### 报告问题
[创建 Issue](https://github.com/langgptai/promptver/issues/new)

### 改进文档
1. Fork 仓库
2. 创建分支: `git checkout -b docs/improve-xxx`
3. 提交改动: `git commit -m "docs: improve xxx"`
4. 推送: `git push origin docs/improve-xxx`
5. 创建 Pull Request

### 提供示例
在 `examples/` 目录添加真实使用案例

### 翻译规范
帮助翻译成更多语言

## 行为准则

- 尊重所有贡献者
- 保持专业和友善
- 专注技术讨论

## 联系

- GitHub Issues
- GitHub Discussions
- LangGPT Community
